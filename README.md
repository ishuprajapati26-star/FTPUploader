# FTP File Uploader

## 📌 Description
A Windows Forms application to upload files to an FTP server with progress tracking and batch processing.

## 🚀 Features
- Folder selection
- Batch file upload
- Parallel processing
- Progress bar UI
- Error handling

## 🛠️ Tech Stack
- C#
- .NET WinForms

## ▶️ How to Run
1. Open project in Visual Studio
2. Configure FTP settings in appsettings.json
3. Run the application
